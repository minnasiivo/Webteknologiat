First chapter

CSS Grid layout on kaksiulotteinen taulukko, johon asetellaan käyttöliittymän komponentit. 'Itemit' asetellaan containeriin. Cointainerille määritellään tila joka käyttää Grid-layoutia. Annetaan riveille ja sarakkeille niiden käytettävissä oleva koko. Gridin solut voidaan varata alueille.

tässä esimerkiksi käytössä viisi riviä ja viisi saraketta
"header header header header header"
"aside aside article article article"
"aside aside article article article"
"aside aside image image image"
"footer footer footer footer footer"

Sisältö voidaan sitten asetella näille alueille
.item-a { grid-area: header; }
.item-b { grid-area: article; }
.item-c { grid-area: aside; }
.item-d { grid-area: footer; }

Välit

Solujen väli voidaan myös määritellä, asettamalla column.cap ja row-cap. Samoin voidaan asetella mihin solun sisältö asettuu vaaka-(justify-items) ja pystytasossa(align-items).

Sisällön asettelu

Sisältö pitää asettaa joko jollekin alueelle tai suluun/soluihin.


.item-a {
grid-column-start: 2;
grid-column-end: five;
grid-row-start: row1-start;
grid-row-end: 3;
}
