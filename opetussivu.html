HTML Perusteita

HTML 5 on nykyaikainen HTML-merkintäkieli, jossa sovellukset toteutetaan HTML, CSS ja Javascript-tekniikoilla.


HTML-sivu rakentuu seuraavista osista:
  • HTML - Html dokumentin juurielementti
  • HEAD - Otsikkotiedot
  • BODY - Dokumentin runko (näkyvä osa)
  • P - Tekstikappale
  • IMG - Kuva
  • A - Linkki
  • BR - rivinvaihto
  • H1,H2,H...- eri tasoiset otsikot
Elementeillä on (yleensä) alku ja loppu.
Elementin alku merkitään esim.: < html>
ja elementti loppu merkintään esim.: < /html>

HTML-sivun kirjoittaminen aloitetaan komennolla: < !DOCTYPE html>
Tätä seuraa html-elementin aloitus ja lopetus tagit < html> ja < / html>
joiden väliin rakennetaan itse html-koodi.

Esimerkki HTML-koodin rakenteesta:

  1. < !DOCTYPE html>
  2. < html>
  3. < head>
  4. < title> Tähän kirjoitetaan sivun nimi < /title>
  5. < /head>
  6. < body> varsinaisen sivun siltö tulee 'body'-tagien sisälle.
  7. < h1>Tässä on ykköstason (eli isoin) otsikko < /h1>
  8. < p>P-tagien sisään kirjoitetaan sivun kappaleet. < /p>
  9. < /body>
  10. < /html>

Yksinkertaisen mallin HTML-rakenteesta löydät myös w3schools.com -sivustolta. Samalla sivustolla pystyt myös harjoittelemaan HTML-koodin kirjoittamista.

Tyhjät elementit

Elementti, jolla ei ole sisältöä (esim. muita elementtejä) kutsutaan tyhjiksi elementeiksi
Tyhjälle elementille ei tarvita erillistä lopputunnistetta, mutta loppuun voidaan lisätä '/>',
jolloin elementistä tulee XHTML-yhteensopiva

Esimerkkejä tyhjistä elementeistä:

Elementti Selite
BR Rivin vaihto (!HUOM! ilman tätä rivinvaihto ei näy käyttäjälle)
IMG Kuva
HR Vaakaviiva (tekstin väliin)
INPUT Syöttökenttä
LINK Linkki toiseen resurssiin
META Metatietoa esim. sivun koodauksesta
